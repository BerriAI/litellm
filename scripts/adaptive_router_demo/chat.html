How it works

Each message goes through the adaptive router which classifies your request type (code, writing, factual…) and uses a Thompson-sampling bandit to pick the model with the best quality for that category.

After 4+ messages, positive feedback signals (✓ in the activity log) update the bandit. Watch the bars move in the live dashboard.
